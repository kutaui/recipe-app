# Recipe app
A basic recipe app that I designed and coded myself. Uses the Recipes REST API that I made.

## Technologies
Project is created with:
* React
* Typescript
* Context API
* React Router
* Material UI

## Recipe API
This is the API that I used. It is open-source and free to use.
